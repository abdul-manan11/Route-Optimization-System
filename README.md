# Route Optimization System

## Overview
Route Optimization System is a desktop application developed using **C++** and **Qt Framework** for visualizing and optimizing routes using graph algorithms. The system enables users to create route networks, connect cities with weighted paths, and compare different optimization techniques to determine efficient routes.

This project demonstrates graph traversal, shortest path algorithms, and route analysis through an interactive graphical interface.

---

## Features

✔ Interactive GUI using Qt  
✔ Graph and city visualization  
✔ Add nodes and weighted routes  
✔ Shortest path calculation  
✔ Multiple algorithm comparison  
✔ Display optimized path and total cost  
✔ Dynamic graph representation  
✔ Route analysis and performance comparison  

---

## Algorithms Implemented

### Dijkstra Algorithm
Calculates the shortest path between source and destination nodes using weighted graphs.

### Greedy Search
Chooses the locally optimal route at each step.

### Brute Force Method
Checks multiple route possibilities to determine paths.

### Manan Algorithm
Custom optimization approach designed for improved route selection.

## Installation

Clone repository:

```bash
git clone https://github.com/yourusername/Route-Optimization-System.git
```

Build project:

```bash
mkdir build
cd build
cmake ..
make
```

Run:

```bash
./RouteOptimizationSystem
```
